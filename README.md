## Sobre o projeto

Projeto desenvolvido afim de testar as habilidades no desenvolvimento Front-end


Desenvolver uma aplicação utilizando a api do GITHUB https://developer.github.com/v3/ consumindo os seguintes endpoints:

Endpoint user: https://api.github.com/users/USER_GITHUB

Endpoint repos: https://api.github.com/users/USER_GITHUB/repos

Endpoint starred: https://api.github.com/users/USER_GITHUB/starred{/owner}{/repo}

 A aplicação deverá constituir três componentes principais:

 O campo de busca.
 Visualização de resultados.
 Dois botões para executar um determinado resultado.
 Ao clicar nos botões de repos e starred, deverá mostrar uma lista simples de cada endpoint apresentado anteriormente.


 Funcionalidades

  Pesquisar por usuários do Github:

 Pesquisar de Usuário
 Exibir todos os Repositorios do usuário
 Exibir os mais visitados
 Mensagem de erro para usuário
Este projeto consiste em três partes:

Desenvolvimento do Frontend das tela pesquisar
Validação do formulário de buscar
Design Responsivo

As seguintes ferramentas foram usadas na construção do projeto:

Front-end: HTML | CSS | JS
